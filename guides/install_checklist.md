# Windows 11 reset checklist
1. Check folders in the Windows side and save important files
    - Documents
    - Images
    - Downloads
1. Update `thunderbird` backup using BORG
2. Update `pictures` backup using BORG
3. Check all git repositories in WSL for pending commits 
4. Save SSH in WSL2 keys as needed
5. Make a full backup of WSL2 `.vhdx` file as needed
6. Check Lightroom Classic library backup (should be in Google Drive)
7. Update Windows 11 and WSL2 configuration guides as you go
